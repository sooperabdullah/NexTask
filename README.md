# NexTask – Service Marketplace Frontend

NexTask is a web-based platform designed to bridge the gap between local service providers (carpenters, plumbers, electricians) and customers. This repository contains the frontend implementation, focusing on a seamless user journey and responsive design.

## 🚀 Project Overview
Finding reliable local help is often a manual and fragmented process. NexTask aims to digitize this experience through an intuitive booking interface.

* **Status:** Frontend Completed | Software Design & Analysis (SDA) Documentation In Progress.

## ✨ Key Features
- **Responsive Landing Page:** Optimized for both mobile and desktop users.
- **Service Categories:** Intuitive navigation for different professional niches.
- **Booking Flow:** A structured interface for users to select services and view professional profiles.
- **User-Centric Design:** Clean UI focused on accessibility and ease of use.

## 🛠️ Tech Stack
- **Languages:** HTML5, CSS3, JavaScript
- **Styling:** Custom CSS
- **Tools:** VS Code
